Place Order
Last Update Date: 13.07.2023

Overview
This API finalizes the booking process by creating a confirmed reservation with traveller details and selected hotel offer. It's the critical step that converts a checked offer into an actual booking. The API requires complete traveller information and must be called before the offer expires to ensure successful reservation creation.

Request URL
http://api.stage.paximum.com/v1/booking/placeorder
Request Model
Attribute	Type	Required	Description
Travellers	Array (Object)	Yes	Travellers of the booking
HotelBookings	Array (Object)	Yes	Bookings
AgencyReferenceNumber	String	Yes	Your reference number
Important Notes:

You must make a request before the offer expired
You must send the room Id we have provided
We can return a response in up to 2 minutes, usually under 1 minute
Confirmed is the only expected status when placeOrder is successful
All required fields are mandatory!
Traveller Object
Attribute	Type	Required	Description
TravellerNo	String	Yes	Traveller number
Type	String	Yes	Traveller type (Adult, Child or Infant)
Title	String	Yes	Title (Mr, Ms, Mrs, Mss, Chd or Inf)
Name	String	Yes	First name
Surname	String	Yes	Surname
BirthDate	String	Yes	Traveller age, optional for adults
isLead	Boolean	Yes	Is lead traveller
Email	String	No	Email address
Phone	String	No	Phone number
Mobile	String	No	Mobile number
Nationality	String	Yes	Nationality code (2 letters)
HotelBooking Object
Attribute	Type	Required	Description
OfferId	String	Yes	Offer ID from search response
ReservationNote	String	No	Customer note for booking
Rooms	Array (Object)	Yes	Room booking details
Room Booking Object
Attribute	Type	Required	Description
RoomId	String	Yes	Room ID from offer
Travellers	Array (String)	Yes	The TarvellerNO of the travellers in the room, these numbers should be defined in request travellers
