Hotel Details
Last Update Date: 13.07.2023

Overview
This API retrieves comprehensive information about a specific hotel. It's used to display detailed hotel information.

Request URL
http://api.stage.paximum.com/v1/search/hoteldetails
Request Model
Attribute	Type	Required	Description
HotelId	String	Yes	This field contains the HotelId as a string.
Request Sample
{
  "hotelId": "100122"
}
Response Model
Attribute	Type	Description
Id	String	Hotel id
Name	String	Hotel name
Description	String	Hotel Description
City	Object	Hotel city
Country	Object	Hotel country
Stars	Double	Hotel star (category)
Rating	Double	Hotel rating
ReviewUrl	String	Review url
Photos	Array (String)	Hotel images
Themes	Array (String)	Hotel themes
Facilities	Array	Hotel facilities
Content	Array (Object)	Hotel content
Address	Object	Hotel address
City Object
Attribute	Type	Description
Id	String	City id
Name	String	City name
Country Object
Attribute	Type	Description
Id	String	Country id
Name	String	Country name
Content Object
Attribute	Type	Description
Title	String	Content title
Description	String	Content description
Address Object
Attribute	Type	Description
AddressLines	Array (String)	Address lines
Street	String	Street name
StreetNumber	String	Street number
CityName	String	City name
CountryCode	String	Country code
Phones	Array (Object)	Phone numbers
Mail	String	Hotel email address
HomePage	String	Hotel website
Geolocation	Object	Geographic coordinates
Phone Object
Attribute	Type	Description
PhoneNumber	String	Phone number
isDefault	Boolean	Is default phone
Type	String	Phone type (voice or fax)
Geolocation Object
Attribute	Type	Description
Longitude	Double	Longitude coordinate
Latitude	Double	Latitude coordinate
