Get Bookings
Last Update Date: 13.07.2023

Overview
This API retrieves a list of bookings based on various filter criteria such as creation dates, check-in dates, or reference numbers. It's essential for tracking multiple reservations, generating reports, and providing customer service. You can filter by date ranges or retrieve all bookings for your account.

Request URL
http://api.stage.paximum.com/v1/booking/getbookings
Request Model
Attribute	Type	Required	Description
From	DateTime	No	Booking creation from date
To	DateTime	No	Booking creation to date
CheckInFrom	DateTime	No	Booking checkin from date
CheckInTo	DateTime	No	Booking checkin to date
AgencyReferenceNumber	String	No	Your reference number
GetAllBookingsOfUserAccount	Bool	No	Send true for get all bookings of user account
Request Samples
{
  "from": "2023-07-13",
  "to": "2023-07-14",
  "GetAllBookingsOfUserAccount": true
}
{
  "checkinfrom": "2016-10-01",
  "checkinto": "2016-11-08",
  "GetAllBookingsOfUserAccount": false
}
{
  "agencyReferenceNumber": "d546d4fdsf6"
}

Bookings	Array (Object)	Requested bookings

checkout	String (DateTime)	Check-out date
amount	Object	Booking amount
status	String	Booking status
notes	Array (String)	Booking notes
orderNumber	String	Order number
bookingNumber	String	Booking number
hotelId	String	Hotel ID
boardCategories	Array (String)	Board categories
cancellationPolicies	Array (Object)	Cancellation policies
nationality	String	Guest nationality
Amount Object
Attribute	Type	Description
Amount	Decimal	Amount value
Currency	String	Currency code
CancellationPolicy Object
Attribute	Type	Description
